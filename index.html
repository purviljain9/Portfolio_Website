Sorting Visualizer

The project is a web-based sorting visualizer application that allows users to observe and understand various sorting algorithms in action. Users can input the size of the array, and the application generates a random array of numbers. The numbers are represented as vertical bars, with the height of each bar corresponding to the value of the element. This visual representation helps users comprehend the sorting process intuitively.
